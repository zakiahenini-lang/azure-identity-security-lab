## Project Title

**Azure Identity Security and Access Governance**

## Short Description

This project is a Microsoft Entra ID security lab focused on identity and access management, authentication, authorization, Conditional Access, MFA, privileged access, and access governance.

The project demonstrates how identity security controls can be designed and documented in a cloud environment.

## CV Bullet Points

- Implemented identity and access management concepts using Microsoft Entra ID.
- Designed users, groups, and role-based access control for a cloud identity environment.
- Configured and documented MFA and Conditional Access policies for privileged access.
- Evaluated privileged access management and access governance controls.
- Created technical documentation covering identity design, security controls, and access review strategy.
